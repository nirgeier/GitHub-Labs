# Lab 01: Working With Repositories

&emsp;⚙️&emsp;This lab guides you through the essential tasks for working with GitHub repositories    
&emsp;⚙️&emsp;You will learn how to create, clone, modify, commit, and push changes to repositories using different methods and tools.

## Lab Contents

### 1. Creating a Repository

#### [Creating a Repository (Web)](01a-Creating-Repo-Web.md)

&emsp;&emsp;📔&emsp;Learn how to create a new GitHub repository using the web interface  
&emsp;&emsp;📔&emsp;Step-by-step guide through GitHub's repository creation form  
&emsp;&emsp;📔&emsp;Configure repository settings, visibility, and initialization options

#### [Creating a Repository (MCP)](01b-Creating-Repo-MCP.md)

&emsp;&emsp;📔&emsp;Create repositories using the GitHub Copilot Model Context Protocol (MCP) extension  
&emsp;&emsp;📔&emsp;Automate repository management tasks directly from VS Code  
&emsp;&emsp;📔&emsp;Use natural language prompts to create and configure repositories

#### [Creating a Repository (gh)](01c-Creating-Repo-gh.md)

&emsp;&emsp;📔&emsp;Master the GitHub CLI (`gh`) for repository creation  
&emsp;&emsp;📔&emsp;Create repositories from the command line with advanced options  
&emsp;&emsp;📔&emsp;Integrate repository creation into your terminal workflow

### [2. Cloning a Repository](02-Cloning-Repo.md)

&emsp;&emsp;📔&emsp;Download repositories from GitHub to your local machine  
&emsp;&emsp;📔&emsp;Use various cloning methods (HTTPS, SSH, GitHub CLI)  
&emsp;&emsp;📔&emsp;Understand the relationship between local and remote repositories

### [3. Making Changes](03-Making-Changes.md)

&emsp;&emsp;📔&emsp;Learn how to modify files in your local repository  
&emsp;&emsp;📔&emsp;Work with Git branches for organized development  
&emsp;&emsp;📔&emsp;Commit and push changes back to GitHub   

## Summary

By completing this lab, you will be able to:

&emsp;&emsp;✅&emsp;Create new repositories using multiple methods (MCP, Web, CLI)  
&emsp;&emsp;✅&emsp;Clone repositories to your local machine  
&emsp;&emsp;✅&emsp;Make and track changes to files  
&emsp;&emsp;✅&emsp;Commit your work with meaningful messages  
&emsp;&emsp;✅&emsp;Push your changes to GitHub for collaboration
