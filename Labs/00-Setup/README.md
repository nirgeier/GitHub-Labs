# Lab 00: Setup 

&emsp;&emsp;⚙️&emsp;This lab will guide you through the essential steps to prepare your environment for working with Git and GitHub.   
&emsp;&emsp;⚙️&emsp;By the end of this lab, you will have the necessary tools installed and your GitHub account configured.  
&emsp;&emsp;⚙️&emsp;After completing this lab, you will be prepared to start working with repositories.

## Lab Contents

### [01. Install Tools](01-Install-Tools.md):

&emsp;&emsp;📔&emsp;Learn how to install Git on macOS, Windows, or Linux.  
&emsp;&emsp;📔&emsp;Optionally install the GitHub CLI (`gh`) for advanced GitHub interactions from your terminal.  
&emsp;&emsp;📔&emsp;Configure your Git identity (name and email) for commit attribution.

### [02. Setup GitHub](02-Setup-GitHub.md):
  
&emsp;&emsp;📔&emsp;Step-by-step instructions to create and verify your GitHub account  
&emsp;&emsp;📔&emsp;Personalize your GitHub profile   
&emsp;&emsp;📔&emsp;Generate and add SSH keys for secure authentication  
&emsp;&emsp;📔&emsp;Add your SSH key to GitHub and test your connection
